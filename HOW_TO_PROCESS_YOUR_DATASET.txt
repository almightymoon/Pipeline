╔══════════════════════════════════════════════════════════════╗
║   📊 HOW TO PROCESS YOUR DATASET - STEP BY STEP   ║
╚══════════════════════════════════════════════════════════════╝

STEP 1: Open the configuration file
────────────────────────────────────
  nano repos-to-process.yaml
  
  (or use any text editor)


STEP 2: Add your dataset repository
────────────────────────────────────
  Delete the example and add your own:
  
  repositories:
    - repo: https://github.com/YOUR-USERNAME/YOUR-DATASET.git
      format: json


STEP 3: Save and push
──────────────────────
  git add repos-to-process.yaml
  git commit -m "Process my dataset"
  git push


STEP 4: Check GitHub Actions
─────────────────────────────
  Open in browser:
  https://github.com/almightymoon/Pipeline/actions
  
  You'll see "🔄 Auto Process Repositories" running!


STEP 5: Download results (after 2-3 min)
─────────────────────────────────────────
  Click on the completed run
  Scroll to "Artifacts"
  Download "processed-repos-[number]"
  
  You'll get:
  ✅ processed.json
  ✅ processed.csv
  ✅ processed.parquet
  ✅ processing-results.json
